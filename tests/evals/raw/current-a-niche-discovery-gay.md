## 唯一安全续跑协议

```text
用户提供合法、可读取的访问条件或由其本人导出的公开研究材料
→ 先记录访问时间、入口、查询词、排序、个性化状态与停止条件
→ notes-first：先打开并核验笔记 URL、标题、正文/逐页内容、发布时间、可见指标与发现路径
→ 对笔记去重，同时保留综合/热门/最新的全部发现路径
→ 从已核验重点笔记反查作者账号；不凭印象先列账号
→ 对重点账号读取近期连续、同类、非置顶的可比笔记
→ 建立同口径近期普通基线，再标记相对高位、低位与不可比异常
→ 主动补独立反例：失败、质疑、不适合、低表现与不同账号样本
→ 同时保留 baseline / high / low / counterexample，不能只看爆款
→ 只有来源、日期、scope、反例和置信度条件闭合后，才评估账号类型与内容模式
→ 选题须绑定可追溯需求/内容证据；证据不足继续研究，不生成标题、封面或成稿
```

恢复访问后若再次出现登录墙、验证码、循环跳转、拒绝访问或风控信号，立即按同一检查点停止，不进行重试或绕过。
